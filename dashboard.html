📊 Dashboard Integral de Rendimiento

Contexto del Proyecto

Este sistema evalúa la fatiga neuromuscular mediante el análisis cinemático de los movimientos del cursor. El sistema matemático compara el rendimiento en tiempo real (score) con las derivadas de posición (velocidad y aceleración) para detectar patrones de error y desgaste cognitivo.

📈 Score por partida

Evolución histórica del rendimiento. Identifica la curva de aprendizaje y estabilidad del usuario.

Análisis Matemático (Carga el CSV)

Velocidad v(t) - 1ra Derivada

Vector velocidad ($ds/dt$). Indica la rapidez de ejecución. Picos altos reflejan eficiencia motriz.

Aceleración a(t) - 2da Derivada

Variación de velocidad ($dv/dt$). Los picos indican temblor (jitter) o ajustes bruscos de precisión.

Monotonía (Estado Creciente/Decreciente)

Estado: 1 (Rendimiento Óptimo) vs -1 (Fatiga detectada). Identifica cuándo el usuario comienza a perder precisión.
