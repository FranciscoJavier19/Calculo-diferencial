📊 Dashboard Integral de Rendimiento

📈 Score por partida

Contexto: Rendimiento acumulado. Evalúa la curva de aprendizaje del usuario durante la sesión.

📉 Derivada (Cambio de rendimiento)

Sistema Matemático: Representa la variación ($\Delta$) entre partidas consecutivas ($P_n - P_{n-1}$). Indica si el usuario mejora o presenta fatiga.

Análisis Matemático (Carga el CSV)

Velocidad v(t) - 1ra Derivada

Contexto: Mide la rapidez del cursor ($ds/dt$). Picos altos reflejan movimientos balísticos precisos.

Aceleración a(t) - 2da Derivada

Sistema Matemático: Cambio en la velocidad ($dv/dt$). Detecta el jitter (micro-temblor) inducido por fatiga muscular.

Monotonía (Estado Creciente/Decreciente)

Contexto: Clasificación binaria (1: Creciente / -1: Decreciente). Define el umbral donde el rendimiento declina.
